Sistemas de cores RGB

O Sistema de cores RGB com profundidade de 24 bits é o sistema de cores utilizado na web. A sigla RGB (Red, Green and Blue) descreve os três canais de frequência que os monitores utilizam para gerar todos as cores que são exibidas através deles. Lembrando que o monitor, bem como as imagens exibidas nele, são formada por pixels e há uma relação entre os pixels do monitor e os pixels das imagens, ou seja, basicamente os pixels do monitor geram os pixels das imagens. Essa composição pode ser visualizada quando se faz a ampliação de uma tela de monitor, onde se encontra diferentes intensidades das cores vermelho, verde e azul para formar toda a paleta de cores que observamos.

As principais cores que precisamos saber do sistema RGB são: vermelho(#FF0000), verde(#00FF00), azul(#0000FF), amarelo(#FFFF00), magenta(#FF00FF), ciano(#00FFFF), branco(#FFFFFF) e preto(#000000). Repare que para alcançar determinada cor é preciso apenas acertar, dentro da base hexadecimal, a intensidade correta de cada um dos três canais básicos.

Aula 05 - Formatação de texto com CSS

O Cascading Style Sheets (CSS), ou seja, Folhas de Estilo em Cascatas. É a maneira mais indicada de formatar/estilizar o código criado por linguagens como HTML, XML ou XHTML.

O CSS permite formatação de tags, isto é, é possível indicar um estilo para uma tag e sempre que ela for utilizada será exibida com essa formatação. Isso proporciona uma padronização para a página e uma facilidade na progrmação.

Segundo Título

É possível trabalhar com o CSS de três formas diferentes: Incorporado, ou seja, abrindo uma tag de estilo no documento; Inline, isto é, abrindo o parâmetro de estilo dentro do próprio comando html; e Externo, onde existe um arquivo destinado ao CSS que posteriormente é linkado ao arquivo HTML.

Terceiro Título

Para formatar uma tag em css, utiliza-se o nome da tag como identificador e os parâmetros a serem formatados aparecem entre chaves {}.

Segue abaixo alguns dos parâmetros que podem ser formatados com CSS:

  • color: Define a cor. Recebe valores em texto (red) ou em hexadecimal (#FF0000).
  • font-size: Define o tamanho da fonte em pixels.
  • font-family: Define o tipo da fonte do texto. É possível trabalhar com mais de uma fonte neste parâmetro. Caso não encontre a primeira fonte especificada, procura pela próxima e assim em diante.
  • letter-spacing: Define o espaçamento entre as letras em pixels utilizando tanto valores positivos como negativos.
  • word-spacing: Define o espaçamento entre as palavras em pixels utilizando tanto valores positivos como negativos.
  • text-decoration: Define uma decoração que pode aparecer no texto, por exemplo, sublinhado.
  • text-align: Define o alinhamento do texto à direta, à esqueda, centralizdo ou justificado.
  • font-style: Define o estilo do texto, por exemplo, itálico.
  • text-transform: Define uma transformação no texto, por exemplo, todas letras maiúsculas, todas as letras minúsculas ou somente a primeira letra de cada palavra maiúscula.
  • text-indent: Define a identação do texto, por exemplo, para o espaçamento do parágrafo, em pixels.
  • padding-left: Define o espaçamento da página à esquerda em pixels.
  • padding-right: Define o espaçamento da página à direira em pixels.

Quarto Título

Ás vezes, deseja-se formartar apenas uma parte do texto sem que atrapalhe a formatação do resto. Para realizar essa tarefa, utiliza-se a tag < span > < / span > que defini uma caixa no ponto em que a formação é desejada e então formata-se essa caixa. Lembrando que span é diferente de div. Uma das diferenças é que a tag span não ocupa um espaço na página.

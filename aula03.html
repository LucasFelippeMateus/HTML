Aula 03 - Listas de definição e Meta Tags

Listas de definição

Diferente das listas ordenadas e não ordenadas, utiliza-se as listas de definição para grandes blocos de textos. Nesse tipo de lista não existem marcadores, pois trabalha-se com a ideia de título e descrição. As listas de definição são construídas através das tags < dl > e < / dl > de definition list, já para os títulos utilizamos as tags < dt > < / dt > de definition title enquanto que para as definições utliza-se as tags < dd > e < / dd > de definition description. Veja um exemplo de lista definição a seguir:

Título 1
Texto referente ao título 1.
Título 2
texto referente ao título 2.

Meta Tags

As Meta Tags são tags de configuração do documento para definiçõe como palavras chaves, descrição, autor, conteúdo, atualização e caracteres.

Como são tags de configuração e definição, as Meta Tags são inseridas dentro do cabeçalho (< head > < / head >).

Uma das categorias das Meta Tags é a name, a qual utiliza-se para definir autor, descrição e palavras chave. Basta indicar qual dessas caracteristicas deseja-se definir e atribuir o seu valor no parâmetro content.

Outra categoria de Meta Tags é aquela destinada ao http. Um dos seus usos é atualizar a página em certos intervalos de tempo através do Refresh, lembrando que além de atualizar também é possível redirecionar para outra página passado determinado tempo. Ainda sobre as Meta Tags voltadas ao http, temos o content type que define a tabela de caracteres utilizados na página.
